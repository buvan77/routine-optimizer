# Routine Optimizer
Full stack routine management system using Spring Boot and React

## Overview

Routine Optimizer is a full-stack web application designed to help users manage and optimize their daily routines. The system tracks user activities, analyzes performance, and dynamically adjusts schedules based on user behavior.

## Features

* Track daily routine activities
* Automatically reschedule missed high-priority tasks
* Generate weekly performance insights
* Adaptive scheduling based on productivity patterns

## Tech Stack

* Backend: Java, Spring Boot
* Frontend: React.js
* Database: PostgreSQL

## Architecture

Frontend (React) communicates with the backend (Spring Boot) via REST APIs. The backend processes business logic and interacts with the PostgreSQL database for data persistence.

## Key Highlight

The application includes intelligent rescheduling logic that adjusts user routines when high-priority tasks are missed. It also provides performance analytics to help users improve consistency.

## Current Improvements

* Refactoring into service-based architecture
* Improving API response structure
* Exploring AI-based recommendation features

## How to Run

### Backend

1. Navigate to the backend folder
2. Run the Spring Boot application
3. Server starts on http://localhost:8080

### Frontend

1. Navigate to the frontend folder
2. Run `npm install`
3. Run `npm start`
4. Application runs on http://localhost:3000
